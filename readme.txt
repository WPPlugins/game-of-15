== Description ==

= Shortcode to add game of 15 =

Useful shortcode to a simple game on your page or posts

[gameof15]

This shortcode add a javascript function that  add a Game of 15 on your posts. You can set a background-image and the color of the numbers.
[gameof15 immagine=“url”] to add an image of background
IMPORTANT: image size: 400x400
[gameof15 color=“#000000”] to change the color of the numbers.
[gameof15 immagine=“url” color=“#000000”] to add an image and change the color.

1. Install Game of 15.
2. Activate Game of 15.
3. Add [gameof15].
